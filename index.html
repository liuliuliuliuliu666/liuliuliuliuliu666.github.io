使用 GitHub + Hexo 搭建个人博客

GitHub创建个人仓库

  • 登录GitHub，点击 GitHub 中的 New repository 创建新仓库，仓库名应该为：用户名 .github.io 。这里用户名是 GitHub 帐号名称，这是固定写法。
  • 我的仓库名为：liuliuliuliuliu666.github.io

    安装Git

  • 在终端进行操作，设置 user.name 和 user.email 配置信息，设置为全局

  • 通过注册的邮箱生成 ssh 密钥文件：

  • 直接三个回车，不需要设置密码，最后得到了两个文件：id_rsa和id_rsa.pub

  • 打开 id_rsa.pub 文件，将里面的内容全部复制，粘贴到GitHub的SSH and GPG Keys的框里
  • 添加好 SSH Key后，进行测试
  • 选择 yes，如果看到Hi后面是你的用户名，就说明成功了

    安装Node.js

  • 可以在https://nodejs.org/en/download/进行对应电脑版本的下载
